Документация

Обзор

Этот документ даст вам общее представление о Disapi. Он предназначен для того, чтобы помочь вам приступить к изучению и разработке собственных инструментов и продуктов.


Коды ответов

API возвращает все данные в допустимом JSON. Некоторые языки программирования включают встроенную поддержку JSON. Для тех, кто этого не знает, вы можете найти подходящую библиотеку на http://www.json.org.


Обратите внимание, что наши API возвращают только непустые значения для экономии пропускной способности. Любые числовые поля или строки, которое не возвращается, могут приниматься равным null.


2XX КОДЫ ОТВЕТА

Для 2xx кодов ответов всегда можно ожидать тело ответа, задокументированное на справочной странице API.

Для кодов ответа, отличающихся от 200, обратите внимание на следующее:

1. Ответ не гарантирует тело ответа.

2. Если есть тело ответа, он не гарантированно будет JSON.

3. В настоящее время мы возвращаем JSON с читаемой человеком информацией.


4XX Коды ОШИБОК

Ошибки 4xx предназначены для указания на то, что клиент не смог предоставить действительный запрос. Ниже приведены наиболее распространенные коды ошибок 4xx, с которыми вы можете столкнуться при использовании API.


(400) Эта ошибка указывает на синтаксическую ошибку в запросе, и поэтому запрос был отклонен.


(403) Эта ошибка указывает на то, что сервер понял запрос, но отказался его авторизовать.


(404) Эта ошибка указывает на то, что сервер не нашел данных для выполняемого запроса API.


(415) Эта ошибка указывает на то, что сервер отказывается обслуживать запрос, поскольку текст запроса находится в неподдерживаемом формате.


5XX КОДЫ ОШИБОК

Ошибки 5xx указывают на то, что сервер знает, что он совершил ошибку или не может выполнить запрос. Ниже приведены наиболее распространенные коды ошибок 5xx, с которыми вы можете столкнуться при использовании API.


(500) Эта ошибка указывает на непредвиденное состояние или ситуацию, которое помешало серверу выполнить запрос API.


(503) Эта ошибка указывает на то, что сервер в настоящее время недоступен для обработки запросов по неизвестной причине.
